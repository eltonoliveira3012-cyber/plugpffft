What ships

  • Instant tray icon with no full app window.
  • Real charger connect detection through Electron power events.
  • Custom fart icon set plus separate plug-in and unplug fart sounds.
  • Installer targets for DMG, ZIP, NSIS, and portable Windows builds.

Install flow

  1. Download the installer for your operating system.
  2. Move the app into Applications on macOS, or run the Windows installer.
  3. Launch PlugPffft once and let it start at login from the tray menu.
